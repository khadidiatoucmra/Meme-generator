# Générateur de Mèmes

Un générateur de mèmes en ligne développé en HTML, CSS et JavaScript.

## Fonctionnalités

- Upload d'images depuis l'ordinateur  
- Ajout de texte en haut et en bas  
- Personnalisation couleur et taille du texte  
- Aperçu en temps réel  
- Téléchargement du mème en PNG  
- Sauvegarde dans la galerie (localStorage)  
- Suppression des mèmes  
- Partage sur WhatsApp, Facebook, Twitter  

## Technologies utilisées

- HTML5  
- CSS3 + Bootstrap 5  
- JavaScript (Vanilla)  
- Canvas API  
- localStorage  

## Structure du projet

meme-generator/
├── index.html
├── style.css
├── script.js
└── README.md  

## Comment lancer le projet

1. Ouvrir le dossier dans VS Code  
2. Installer l'extension Live Server  
3. Clic droit sur index.html  
4. Cliquer sur "Open with Live Server"
